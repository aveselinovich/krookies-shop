# Локальный вход

В этой версии вход для локальной разработки работает как раньше:

- админ: `+79959178862`
- код: `1111`

Для обычного клиента можно использовать любой российский номер и код `1111`.

Хеш пароля администратора теперь хранится только в базе.
`ADMIN_PASSWORD` можно использовать только для первичного `seed`, но логин сотрудников сравнивает пароль только с `users.passwordHash`.

После изменения Prisma-схемы нужно один раз обновить структуру базы:

```bash
npx prisma db push
npx prisma generate
npm run dev
```

Если запускаете проект впервые:

```bash
createdb krookies
npx prisma migrate dev --name init
npx prisma db seed
npm run dev
```
